# Sapore è Sapere — Restaurant Knowledge Base

This skill holds the single source of truth for all content about the restaurant
**Sapore è Sapere**. Pull names, prices, hours, and copy from the reference files
below rather than inventing them, so the website stays consistent across pages.

## Identity at a glance

- **Name:** Sapore è Sapere
- **Tagline:** Cucina creativa e piatti originali per un'esperienza gastronomica fuori dal comune
- **Opened:** 23 May 2019
- **Cuisine:** Primarily fish/seafood, with vegetarian options; creative, innovative pairings
- **Style:** Elegant but informal — suitable for couples, families, groups, and corporate dinners
- **Location:** Via Marcello Mastroianni snc, 03100 Frosinone (FR), Italy
- **Founders:** Chef Marco Taglione & pastry chef / maître Marika Urbani
